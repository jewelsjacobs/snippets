# C Syntax != JavaScript

A compendium of C syntax that is different that JavaScript.

### Data Operators
Mainly used with pointers

Operator | Description
:---: | ---
`sizeof()` | Get the size of
`[]` | Array subscript
`&` | The address of
`*` | The value of
`->` | Structure dereference
`.` | Structure reference

### Bitwise Operators

Operator | Description
:---: | ---
`&` | Bitwise and 
`|` | Bitwise or 
`^` | Bitwise xor
`~` | Bitwise one's complement
`<<` | Bitwise shift left
`>>` | Bitwise shift right
